Linux FPAC-HOWTO, Amateur Radio.

Jean-Paul ROUBELAT, F6FBB, jpr@f6fbb.org

v1.0, 24 September 1998

The Linux Operating System has a kernel buil-in protocol named ROSE. FPAC is a set of tools to configure and interface ROSE. This document aims to describe how to install and configure FPAC.
